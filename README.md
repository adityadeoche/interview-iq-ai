# InterviewIQ AI

AI-powered interview preparation and candidate screening platform.

## Features
- **5-Round Interview Engine**: Powered by **Groq llama-3.3-70b-versatile**.
- **Role-Based Portals**: For Candidates, Recruiters, and Campus TPOs.
- **DNA Reporting**: Real-time analysis and skill matrix.

## Tech Stack
- **Framework**: Next.js 15+
- **Styling**: Tailwind CSS
- **Database**: Supabase
- **AI**: Groq SDK (groq-sdk)

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Configure `.env.local` with `GROQ_API_KEY` and Supabase credentials.
4. Run locally: `npm run dev`.
